# Portfolio — Kuete Kamga Starline Lecool

Persönliche Portfolio-Website zur Präsentation meiner Fähigkeiten, Projekte, Erfahrungen und Ausbildung in der Webentwicklung.

## Überblick

Modernes Web-Portfolio mit Glassmorphism-Design, Dark/Light-Mode, mehrsprachiger Unterstützung (Deutsch / English / Français) und fortgeschrittenen CSS-Animationen.

## Funktionen

- **Flüssige Navigation** mit responsivem Menü und animiertem Scrollen
- **Typewriter-Effekt** auf dem Haupttitel
- **Skills-Karussell** mit Endlosscrolling
- **Projektkarussell** mit Maus-, Touch- und Tastaturunterstützung
- **Dark / Light Mode** mit localStorage-Persistenz
- **Mehrsprachig**: Deutsch, Englisch, Französisch (200+ Übersetzungsschlüssel)
- **Erfahrungs-Karten** auf- und zuklappbar
- **Visuelle Ausbildungs-Timeline**
- **Zertifikate-Viewer** über Modals
- **Kontaktformular** via Formspree
- **Scroll-Animationen** mit Intersection Observer API
- **100% responsives Design** (Mobil, Tablet, Desktop)

**Integrationen**

## Projekt starten

Keine Abhängigkeiten erforderlich. Direkt im Browser öffnen:

```bash
# Option 1 — Datei direkt öffnen
open index.html

# Option 2 — Lokaler Server (empfohlen, um CORS-Einschränkungen zu vermeiden)
npx serve .
# oder
python -m http.server 8080
```
